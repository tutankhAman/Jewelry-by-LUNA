## Technical Stack
- Frontend: React.js, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, OAuth
- Payment: Stripe
- Image Storage: Cloudinary
- Deployment: Vercel (frontend), Render (backend)
